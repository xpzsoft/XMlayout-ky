Panel 1

In panel 1, two features has been set using xmlayout.js as follow :

1, ZOOM_OUT is disabled by setting. 

	xm.disablePanelItem(panels[0], xm.RPID.ZOOM_OUT, true);


2, A self-defined right panel button 'user-defiend' is added into right panel. 

	var id = xm.addRightPanelItem(panels[0], "user-defiend", function(e){
        alert("user-defiend click");
        xm.removeRightPanelItem(panels[0], id);//remove the self-defined right panel button
    });
